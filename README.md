## 📝 Writeup Philosophy

Every writeup in this repo follows a consistent structure:

1. **Enumeration** — What did I find and how?
2. **Initial Foothold** — The vulnerability and how I exploited it
3. **Privilege Escalation** — Path from user to root/admin
4. **Lessons Learned** — What this taught me and what to review

> HackTheBox writeups are only published **after** the machine/challenge is retired, in accordance with platform rules.
